# Preset: brutalist — AI 喂养片段

> 用法：项目用 brutalist 风格时，把本片段连同 `DESIGN.md` 一起放进 AI 上下文。

## 一句话
锋利、自信、记忆点强的极简硬朗设计。0 圆角 + 硬边偏移阴影 + 粗黑边 + 高饱和单色。

## 适用
开发者工具、技术博客、个人项目、想要强烈辨识度的产品。

## 颜色（亮色，OKLCH）
| 角色 | 值 | 直觉 |
|---|---|---|
| 背景 | `oklch(1 0 0)` | 纯白（对比拉满，唯一允许纯白的 preset） |
| 正文 | `oklch(0.15 0 0)` | 近纯黑 |
| 边框 | `oklch(0.15 0 0)` | 粗黑边 = 近纯黑 |
| 主色 accent | `oklch(0.85 0.18 100)` | 电光黄 |
| CTA | `oklch(0.15 0 0)` | 黑底白字 |

可替换主色：正红 `oklch(0.55 0.22 25)`。

## 字体
- 英文标题：**Space Grotesk**（700+ 超粗）
- 英文正文：Space Grotesk / 等宽
- 中文标题：**得意黑 Smiley Sans**（免费超粗 display，天生硬朗）
- 中文正文：思源黑体 Medium
- 注意：本 preset 标题允许更粗（display 字体本身的粗，非 font-weight 堆叠）

## 质感（招牌）
- **圆角全 0**（DNA-2 在此特化为直角）
- **硬边偏移阴影**：`4px 4px 0 #000`（无 blur，这是核心识别特征）
- 粗黑边框 1.5–2px，是主要视觉语言
- 高饱和单一强调色，大胆间距对比

## 招牌动作（写出来要像这样）
- 卡片/按钮：直角 + 粗黑边 + 硬边偏移阴影
- hover 时元素**位移**（`translate`）而非渐变，阴影偏移量变化
- 强调色只用一种，用在关键处
- 大字重标题，字号夸张

## 参考点
像 **Gumroad 改版**、**Figma 早期 marketing**、**brutalist web 设计**。
**不像** 柔和 SaaS、毛玻璃、渐变、圆润卡片。
